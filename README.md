# TaskMind

A system for managing goals and automatically breaking down tasks using AI.

# 🌐 Frontend - TaskMind Web UI

This is the frontend for the TaskMind application, built with [Next.js](https://nextjs.org/) and managed with [pnpm](https://pnpm.io/).

## 📦 Getting Started

1. **Navigate to the project directory**:

```bash
   cd web-ui
```

2. **Install dependencies**:

```bash
   pnpm install
```

3. **Start the UI**:

```bash
   pnpm run dev
```
